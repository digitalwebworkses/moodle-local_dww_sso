# DWW Moodle SSO

![Version](https://img.shields.io/badge/version-1.0.0-green)

![Moodle](https://img.shields.io/badge/Moodle-4.x-orange)

![PHP](https://img.shields.io/badge/PHP-8.x-blue)


Secure Single Sign-On plugin for Moodle.

DWW Moodle SSO works together with the DWW Moodle Bridge WordPress plugin to provide automatic and secure authentication between WordPress, WooCommerce and Moodle.

## Features

- Secure SSO token validation
- Automatic Moodle login
- Direct course access
- Shared secret authentication
- Moodle-side logging
- Friendly error handling
- WordPress integration support

## Requirements

- Moodle 4.x
- PHP 8.x

## Installation

1. Download the latest release ZIP.
2. Install the plugin in Moodle:
   Site administration → Plugins → Install plugins
3. Configure the shared secret:
   Site administration → Plugins → Local plugins → DWW SSO
4. Use the same secret configured in DWW Moodle Bridge for WordPress.
